规则介绍

从5.24-5.30号止，每天飞扬数码管家都会发送100-10000金额不等的红包。


活动规则如下：

1、活动紧紧局限于广州区域，系统会自动定位您的位置是否在广州。


2、我们每天会在以下时间段发送红包：

1）上午10：00  

2）中午12：00

3）下午16：00

4）下午18：00

5）晚上10：00


3、每个红包只能抢一次，金额随机发送。


4、每个用户可以发展小弟，小弟抢到的红包会上供20%给你。


5、红包金额满一元可以提现。（这是微信官方对企业红包的限制）提现将立刻达到你的微信零钱包上。
